<p>To use MoneyPress : Amazon Edition you only need to add a simple 
shortcode to any page where you want to show a list of Amazon Products.  
For example:<br/>
<code>[MP-AMZ]</code><br/>
<br/>
You can change the default product categories, number of items returned, and
other display paramters via the plugin settings page and via attributes set
in the shortcode.
